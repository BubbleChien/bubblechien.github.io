UNI PROJECTS
DrKanban

A web application to manage... kanban boards.



Single user mode is based on the LocalStorage API, and the multi-user mode works via Node.js and WebSockets.

The single user version is there, it hasn't been tested for a while though.

Pixie

Pixel art sprite editor for Android.



Support for animation sheets, onion skin and tiling.

You can download the APK, but keep in mind it only has been tested on my Galaxy S so there might be issues.



GAME JAMS
Robestron

A physics sandbox game where you must come up with the craziest way to decapitate Robestron.



Written in C++, using SFML2 and Box2D.

Created for the OWAG 2013.

Azure

A puzzle game about imbalances and perception in relationships.



Made with Unity3D, Blender, Photoshop, SFXR, FL Studio and Audacity.

Created for the 30th Ludum Dare 48h Solo Comp.

Bathyal Commodore

An old-school submarine simulation game, I made the graphics, audio and dialogs.



Used Photoshop, FL Studio and Audacity.

Created for the 31st Ludum Dare 72h Jam.
